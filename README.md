# 📦 Logistik Bot – Telegram uchun aqlli yuk e'lonlari bot

Logistik Bot – bu Telegramdagi logistika bilan bog‘liq e'lonlarni avtomatik aniqlab, ularni standart shablonga keltiradigan aqlli botdir. U foydalanuvchi e'lonlarini aniqlaydi, telefon raqamni yashiradi va tugma orqali ko‘rsatadi. Latin yozuvlarini avtomatik ravishda kirill yozuviga o‘giradi.

---

## ⚙️ Xususiyatlar

- 📌 Guruh, kanal yoki shaxsiy chatda ishlaydi
- 🧠 Logistika bilan bog‘liq xabarlarni aniqlaydi (kalit so‘zlar asosida)
- 🔤 Lotin yozuvini avtomatik Kirillga o‘giradi
- 🧾 E’lonlarni tartibli va chiroyli shablonga joylashtiradi
- 🧹 Original xabarni o‘chiradi (kanal va guruhlarda)
